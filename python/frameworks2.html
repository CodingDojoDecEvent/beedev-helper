Getting started with Flask

Installation and setup

Create a directory for your project and move into it 

    $ mkdir flaskproject
    $ cd flaskproject

Create a virtual environment 

    $ python3 -m venv venv

Activate the environment 

    $ source venv/bin/activate

Install Flask 

    (venv) $ pip install flask

Create a file named server.py and add the following code 

    from flask import Flask

    app = Flask(__name__)


    @app.route('/')
    def hello_world():
        return 'Hello World!'


    if __name__=="__main__":
        app.run(debug=True)

We can run our Flask app with the following command 

    (venv) $ python server.py

Open localhost:5000 in your web browser. You should now see the “Hello, World!” text we added in our route. Congratulations!
