Python Basics

Variables

A variable is a box where you can store the value of a data type in your computer. The information saved in the box can be recalled when the variable name is called. The variable name must follow a naming convention rules in order to be properly retrieved.

Python Data Types

Here is a brief description of each data type:

int: represents integer values, such as 1, 2, 3, etc.

float: represents decimal values, such as 1.0, 2.5, 3.14159, etc.

str: represents a sequence of characters, such as “hello”, “world”, etc.

list: represents an ordered collection of elements, such as [1, 2, 3], [“apple”, “banana”, “cherry”], etc.

tuple: represents an ordered, immutable collection of elements, such as (1, 2, 3), (“apple”, “banana”, “cherry”), etc.

dictionary: represents a collection of key-value pairs, such as {“name”: “John”, “age”: 30}. fruit_baskets={ "basket1"="apples", "basket2"="bananas", "basket3" ="cherries"}

Boolean: A way to compare or assign a variable to True or False. In Python those keywords must be written in uppercase. Also a zero or null or an empty string (" ") is considered False. A 1 or any other value that is not zero or null or an empty string is considered True.
